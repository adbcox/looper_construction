# Looper Construction — Homepage Foundation (Desktop-first)

A clean Vite + React + Tailwind site with react-router.
Built as a calm, editorial homepage foundation you can mark up and iterate on.

## Netlify
- Build: `npm run build`
- Publish: `dist`
- SPA routing is enabled via `netlify.toml`

## What you’ll change first
- `src/pages/Home.jsx` (homepage layout/sections)
- `src/content/siteContent.js` (copy labels)
- `public/images/*` (swap placeholders for real images)
